  • Base de Datos - Frutería
    • Diseñé una base de datos relacional para una pequeña frutería para ayudar a administrar el inventario, las ventas y los clientes de la tienda de manera organizada y eficiente.
    • El sistema habilitó la identificación de productos con mayor demanda durante cada temporada de ventas, lo que permitió un mejor inventario planificación y optimización de recursos, evitando situaciones de escasez o exceso de producto.
    • Mejoró la eficiencia y rentabilidad de la gestión de pedidos de productos agrícolas, brindando información precisa y información en tiempo real que permite la toma de decisiones basada en datos y reduce los riesgos de errores humanos
